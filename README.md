# Hajiki_GMM
## このページについて
このページは松井広信2019「中世土師器皿の楕円フーリエ解析」『中近世陶磁器の考古学』11　雄山閣,37-54頁で使用したデータセットの一部を公開する目的で作成しました。

筆者がRのパッケージmomocsを使用するに当たって、気がついたことなども記載していくので、英語が苦手な日本の考古学研究者に役に立つと思います。
